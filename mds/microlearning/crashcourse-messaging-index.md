	<h2 class="title">Crash Course Messaging</h2>
    <p>
       Tn this course you will get to know the eMagiz platform from a Messaging perspective. Messaging is a powerful pattern that can handle many integration challenges. In this course you'll visit the messaging specific items that you need on top of the platform crash course. So that you can create your first messaging solution to a go-live state. You'll find the proper exercises for each topic, and you can relate the 5 Integration Lifecycle Management phases in the platform. The course contains small exercises to learn each subject, and can act as a reference as well.  
        </br></br>
        - Total time for first time completion: between 60 and 90 minutes
        </br>
        - Last updated - February 2021
    </p>
