	<h2 class="title">eMagiz Use Case - Data retrieval file-based system</h2>
    <p>
       In this course, you will find the collection of the relevant Microlearnings that you can refer to when retrieving data from a file-based system. You can study these microlearnings in the order as presented here to learn in one go all about retrieving data from a file-based system in eMagiz. Note that the referenced microlearnings are identical to those under the Microlearning section.
        </br></br>
        - Total time for first time completion: between 20 and 30 minutes
        </br>
        - Last updated - Q1 2022
